# Picture-It-Dictionary ("PicDict")
"Imagine/picture it (a concept), search for words"

App for communication, speaking.

1. (An) image(s) is added...
   - < (less than) 32x32 : *2* x 2, *3* x 3, (*4* x 4 / *5* x 5)
2. Write it to file??
3. Search words for it/this.
